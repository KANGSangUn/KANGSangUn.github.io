# 새로운 프로젝트 "Meshi Meshi(メシメシ)"

새로운 팀 KKJ
-

Meshi Meshi란?
-
작년 여름 후쿠오카에서 기획한 프로젝트로,  
실제의 명함을 없애고 명함에 대한 모든 것을  
스마트폰으로 해결하겠다는 취지의 새로운 개념의 SNS서비스 입니다.  

사용 기술
-
Android - Kotlin  
IOS     - swift  
Backend - Ruby on Rails  
Database - MongoDB
